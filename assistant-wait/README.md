## Utilisation

On appelera la commande `wait_` suivi par un nombre et le texte-temps.

### Exemples

  - `wait_5 minutes` : attendre 5 minutes
  - `wait_30 secondes` : attendre 30 secondes
  - `wait_2 heures` : attendre 2 heures
